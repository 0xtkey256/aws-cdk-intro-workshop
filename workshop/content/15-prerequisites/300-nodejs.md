The AWS CDK uses Node.js (>= 8.12.0).

* To install Node.js visit the [node.js](https://nodejs.org) website.

* If you already have Node.js installed, verify that you have a compatible version:

    ```s
    node --version
    ```

    Output should be >= 8.12.0:

    ```
    v8.12.0
    ```
